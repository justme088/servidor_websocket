1. Para instalar las librerias, debe iniciar setup.bat
2. Para iniciar el servidor websocket, se usa run_server.bat, que este ejecuta a app.py
3. El archivo data_testing.bat ejecuta el script data_input.py, que despliega una serie de opciones para interactuar con los datos de la base de dato y se pueda ver el cambio.
4. El documento index.html es un cliente demo que se conecta al servidor websocket, el html puede ser ejecutado directamente desde cualquier navegador actual.
